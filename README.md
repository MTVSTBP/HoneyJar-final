## [1] About the Project

`🌟 우리의 서비스가 당신의 일상에 달콤함을 더할 수 있다면.`
1. **주변 맛집 및 착한 가게 알림 제공:** 사용자 위치 주변의 맛집과 착한 가게를 제공하여 보다 효율적으로 확인 및 소비 부담 완화 기대
2. **착한 가게 검색 기능 제공:** 사용자가 원하는 조건에 맞게 검색할 수 있도록, 검색 및 필터링 기능을 활용하여 알맞은 착한 가게 제공
3. **커뮤니티 공간:** 사용자 간 소비에 도움이 되는 다양한 정보를 공유할 수 있는 소통의 장 제공
4. **꾸준한 서비스 업데이트 제공:** 일회성 서비스가 아닌 지속적으로 업데이트하여 사용자들의 알뜰한 소비에 도움을 제공
